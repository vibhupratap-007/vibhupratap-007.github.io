Latest Project

Customer Churn & Salary Prediction Using Deep Learning

Developed a deep learning-based Customer Churn Prediction model achieving 91% accuracy. Designed with data preprocessing and ANN to identify customers likely to leave. Deployed via Streamlit for real-time user input and instant predictions.

LangChain Projects

Built AI applications using LangChain for prompt engineering and LLM integration. Connected large language models with external data sources to enable context-aware responses. Designed modular workflows for chatbots, query answering, and intelligent automation.

Next Word Prediction Using LSTM

Developed an LSTM-based deep learning model for next-word prediction in text sequences. Trained on sequential language data to capture context and generate accurate predictions. Implemented using TensorFlow/Keras with strong results in natural language modeling.

Face Identification and Detection

Engineered a real-time face authentication app using Python, Flask, and JavaScript. Implemented webcam-based image capture with a facial recognition pipeline achieving 85% accuracy. Optimized for minimal latency (200ms), cross-browser compatibility, and seamless accessibility.

Shreyanshi Garg Portfolio Website

Developed a responsive personal portfolio website using HTML, CSS, and JavaScript. Designed to showcase projects, skills, and achievements with a clean modern UI. Includes interactive sections, responsive layouts, and smooth navigation for all devices.

Text Summarizer Using Spacy

Implemented a text summarization tool using spaCy for efficient NLP processing. Applied extractive summarization techniques to condense large texts into key sentences. Designed with Python and spaCy to provide fast, accurate, and meaningful summaries.
